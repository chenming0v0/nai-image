# 更新总结

## ✅ 已完成的功能

### 1. 🔔 智能询问对话框
当用户在 Gallery 模式下已有 1 张图片时，再上传第二张图片会弹出对话框询问：
- **替换参考图**：用新图片替换原有的参考图
- **添加为 Vibe**：将两张图片都变为 Vibe 参考

### 2. 🏷️ 修复图片标签显示

**修复前（错误）**：
```
图片1: "改图"    ❌ 误导
图片2: "参考1"   ❌ 让用户以为可以同时有改图和Vibe
图片3: "参考2"
```

**修复后（正确）**：
```
只有1张图时:
图片1: "改图" 🔵

有2张及以上时:
图片1: "Vibe 1" 🟡
图片2: "Vibe 2" 🟡
图片3: "Vibe 3" 🟡
```

## 🎯 核心原则

在 NAI Gallery 模式中：
- **1 张图 = 改图模式**（img2img，可添加遮罩）
- **2+ 张图 = 全部 Vibe 模式**（风格参考）
- **不能混用**：要么改图，要么 Vibe
